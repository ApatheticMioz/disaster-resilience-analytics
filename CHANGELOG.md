# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - Initial Archival Release

### Added
- Comprehensive ETL pipeline for disaster resilience analysis (`build_unified_dataset.py`)
- Data quality diagnostics and validation framework (`run_diagnostics.py`)
- Integration of 13 heterogeneous datasets (EM-DAT, World Bank WDI, ND-GAIN, etc.)
- Derived indices: DII (Disaster Impact Index), RRS (Resilience Recovery Score), CRI (Composite Resilience Index)
- Interactive Tableau dashboard with 5 coordinated analytical views
- Project documentation and technical report

### Data Sources Integrated
- ND-GAIN (Climate Resilience Index)
- Harmonized Nighttime Lights
- GDACS (Global Disaster Alerts)
- EM-DAT (Disaster Impact Data)
- IMF WEO (Macroeconomic Indicators)
- World Bank WDI (Development Indicators)
- HDR (Human Development Index)
- WGI (Governance Indicators)
- INFORM Risk Index
- FTS (Humanitarian Funding)
- DesInventar (Granular Disaster Losses)
- Barro-Lee (Educational Attainment)
